## 传参说明

### Props参数

| 参数名     | 说明       | 是否必填 | 参数类型      | 默认值     |
| ------- | -------- | ---- | --------- | ------- |
| size    | 图标大小     | 否    | string \| | Number  |
| color   | 颜色       | 否    | string    | #ffffff |
| iconStr | svg图标字符串 | 是    | string    | -       |
