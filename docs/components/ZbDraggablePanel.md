## 传参说明

### Props参数

| 参数名     | 说明                                 | 是否必填 | 参数类型              | 默认值 |
| ------- | ---------------------------------- | ---- | ----------------- | --- |
| columns | 元素数组 每个元素中必须包含id元素, 且id不能相同,否则可能报错 | 是    | [{id,.....},....] | {}  |
